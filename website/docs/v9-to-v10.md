v10.0.0 Migration Guide

> FYI: v10 drops support for Node 20. If you need support for versions of Node that have reached EOL, please use v9.

Please see the [release notes](https://github.com/tclindner/npm-package-json-lint/releases/tag/v10.0.0) for additional changes introduced in v10.0.0.
