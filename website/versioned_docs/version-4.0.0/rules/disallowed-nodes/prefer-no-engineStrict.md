Enabling this rule will result in an error being generated if `engineStrict` is present.

## Example .npmpackagejsonlintrc configuration

```json
{
  "rules": {
    "prefer-no-engineStrict": "error"
  }
}
```

## Rule Details

  "engineStrict": "dummy-value"
